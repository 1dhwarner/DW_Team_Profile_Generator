# DW_Team_Profile_Generator

CLI app that uses NodeJS and Inquirer. Includes passing unit tests.

## Installation:

1. clone the repo
2. npm i
3. npm run test
4. node index.js

A "team.html" file will appear in the dist folder. From there you can view the page in your browser.
